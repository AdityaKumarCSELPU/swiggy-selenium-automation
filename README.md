# swiggy-selenium-automation
Automation testing project that simulates online food ordering on Swiggy using Selenium WebDriver, Java, and Maven.

# Swiggy Food Ordering Automation using Selenium WebDriver

## Project Overview
This project automates the process of ordering food on the Swiggy website using Selenium WebDriver with Java and Maven. The automation script simulates a real user interaction by navigating through the website, searching for restaurants, adding items to the cart, and proceeding to the payment page.

This project was developed as part of a Software Testing Automation assignment.

---

## Technologies Used

- Java
- Selenium WebDriver
- Maven
- WebDriverManager
- Eclipse IDE
- Chrome Browser

---

## Features Automated

The automation script performs the following tasks:

1. Opens the Swiggy website
2. Logs into the account (OTP entered manually)
3. Validates the page title and URL
4. Enters delivery location
5. Searches for a restaurant
6. Selects a food item from the restaurant menu
7. Adds the item to the cart
8. Increases the quantity of the selected item
9. Adds a delivery address
10. Navigates to the payment page (without completing payment)
